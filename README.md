# Sorting Visualizer

A web application showcasing the inner workings of various sorting algorithms.

## Author / Owner
**Sresth khandelwal**

## Implemented Algorithms
1. Bubble Sort
2. Selection Sort
3. Insertion Sort
4. Merge Sort
5. Quick Sort
6. Heap Sort

## Features
1. **Colored Representation of Execution Steps:**
   - **Blue:** Default state
   - **Yellow:** Elements being compared
   - **Red:** Elements identified as in incorrect position and to be moved
   - **Green:** Elements in their correct, sorted position
2. **Visualization Controls:**
   - **Speed of Visualization:** 5 different speed levels to slow down or speed up the animation.
   - **Data Size:** A slider to adjust the size of the array (number of elements).
   - **Generate New Data:** Randomly generates a new array of the selected size.
3. **Complexity Information:** 
   - Displays both **Time Complexity** (Worst, Average, Best case) and **Space Complexity** for the algorithm currently being visualized.

## Usage
Simply open `index.html` in your web browser. Adjust the array size, choose your visualization speed, select a sorting algorithm, and watch the visualization!
